## Requirement 
- java jdk 11, docker and docker-compose
## Run project with docker
1. Clone project from github
```bash
$ git clone git@github.com:Phambanam/mini_blockchain.git
```
2. CD into project folder
```bash
cd mini_blockchain
```
3. Run project with docker compose
```bash
docker-compose up 
```
